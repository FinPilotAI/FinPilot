## **FinPilot: LLM 기반 비즈니스 문서 작성 어시스턴트**  
> 비즈니스 문서 작성을 쉽고 효율적으로 도와주는 스마트 가이드!

`FinPilot` 프로젝트에 오신 것을 환영합니다! 
이 프로젝트는 Google Docs 환경에서 사용자가 더 효율적으로 문서를 작성하고 관리할 수 있도록 설계된 LLM 어시스턴트 서비스를 개발하는 것을 목표로 시작되었습니다. `Chrome Extension`과 `LangGraph` 기반의 LLM 애플리케이션인 `FinPilot`을 통합하여 문서 초안 작성, 단락 생성, 요약 및 확장,  데이터 분석/시각화 등의 기능을 제공합니다.

## 🔍 About FinPilot
FinPilot은 LLM을 활용해 Google Docs 환경에서 문서 작성을 지원하는 Chrome 확장 프로그램입니다. Google Docs 내에서 FinPilot이 제공하는 문서 작성 및 개선 기능을 사이드바를 통해 손쉽게 활용할 수 있습니다. 다양한 기능을 하나의 인터페이스에서 활용할 수 있도록 설계하였습니다.

비즈니스 문서를 작성하고 관리하는 데 필수적인 기능을 다음과 같이 제공합니다:  

1. **초안 작성**: 사용자 입력을 기반으로 구조화된 초안을 자동으로 생성하여 시간을 절약합니다.  
2. **단락 생성**: PDF 또는 최신 웹 데이터를 기반으로 새로운 단락을 생성하여 문서를 강화합니다.  
3. **요약 및 확장**: 문서의 길이를 조정할 수 있도록 요약하거나 확장합니다.  
4. **웹 데이터 기반 시각화**: 실시간 웹 데이터를 수집하고 시각적 인사이트를 제공합니다.  
5. **업로드 데이터 기반 시각화화**: CSV 파일을 업로드하여 자동 분석 및 시각화를 수행합니다.

## 🚀 Key Functions

### '초안 작성'
사용자가 요청한 문서의 초안을 작성합니다. FinPilot은 사람의 문서 작성 과정에 착안하여 문서에 대한 목차를 구성한 후, 각 목차에 대한 내용을 작성해나가는 방식으로 초안을 작성합니다. 각 목차의 내용을 작성하는 과정에서 필요에 따라 주식, 재무제표, 뉴스, 웹 데이터 등의 자료를 스스로 판단하여 수집하며, 이를 바탕으로 작성합니다. 초안 작성의 경우 3~6분 정도 시간이 소요되며, Google Docs 기준 대략 5~10 페이지 분량의 문서를 작성합니다.

### '단락 생성'
특정 주제에 대한 단락을 생성합니다. 작성된 초안에 대해 내용을 보완하고 싶을 때 유용하게 사용될 수 있습니다. 단락 생성 과정에서 참조하기를 원하는 PDF 파일을 업로드하면 FinPilot은 사용자에 요청에 대해 해당 문서의 내용을 참조하여 생성합니다. 만약 PDF 파일이 제공되지 않아도, 웹 검색을 통해 스스로 관련된 자료를 찾아서 적절한 단락을 생성합니다. 내부적으로 Self-RAG와 Corrective RAG의 아키텍처를 채택하여 환각 현상을 방지하고 요청에 대한 적절한 답변을 제공합니다.

## ⚡How to Get Start in Local
### Chrome 확장 프로그램 테스트
1. 이 저장소를 다운로드하거나 클론합니다.
2. Chrome에서 `chrome://extensions/` 페이지를 엽니다.
3. **개발자 모드**를 활성화합니다.
4. **압축 해제된 확장 프로그램 로드** 버튼을 클릭하고 `FinPilot` 폴더를 선택합니다.
5. 확장 프로그램을 고정하여 사용을 시작합니다!

### FinPilot API 테스트
1. OpenAI API 키를 발급받습니다.
2. `src/sidebar.js` 파일에 API 키를 입력합니다.
3. 확장 프로그램을 다시 로드한 후 AI 기능을 테스트합니다.
