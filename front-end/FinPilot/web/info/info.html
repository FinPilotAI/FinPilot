초안 작성


사용자가 요청한 문서의 초안을 작성합니다. FinPilot은 사람의 문서 작성 과정을 착안하였습니다. 먼저 문서의 섹션(목차)를 구성하고, 각 섹션의 내용을 작성하는 과정에서 필요한 자료를 스스로 판단하여 수집합니다. 주식, 재무제표, 뉴스 등의 자료를 수집하며 수집한 자료를 분석해 각 섹션의 내용에 충실하도록 작성합니다. 모든 섹션의 내용을 작성하면 내용을 병합해 하나의 완성된 문서를 사용자에게 제공합니다. 작성 과정에서 참조한 모든 출처를 사용자에게 제공하여 문서 내용을 검증하고 이를 명시할 수 있도록 지원합니다.


사용 방법:

  • "초안 작성" 옵션을 선택합니다.
  • 채팅창에 초안을 작성하고 싶은 주제를 입력합니다.

    (예: "삼성전자 기업 분석 보고서 초안 작성" 또는 "현대자동차")

  • 메시지를 전송하면 주제에 적합한 초안을 제공합니다.
  • 예상 소요 시간: 5분 내외

예시:

  • 입력: "삼성전자 기업 분석 보고서 초안 작성"
  • 출력:

    • "1. 삼성전자 개요 삼성전자는 대한민국을 대표하는 글로벌 전자 기업으로, 반도체, 모바일, 디스플레이 등 다양한 분야에서 세계적인 경쟁력을 보유하고 있습니다. 최근 삼성전자는 2025년 2월 7일에 갤럭시 S25 시리즈를 전 세계에 출시하며... "

단락 생성


사용자가 요청한 특정 주제에 대한 단락을 생성합니다. 보통 초안 작성 후 해당 문서의 내용을 보완하는 과정에서 사용할 수 있습니다. PDF 업로드 시, 해당 문서의 내용을 기반으로 단락 생성이 가능합니다. PDF가 제공되지 않아도, 웹에서 관련 정보를 수집해 단락을 생성할 수 있습니다. 수집된 자료를 단순히 사용하는 것이 아닌 사용자 요청에 대한 관련성을 측정하여 관련성 높은 문서 만을 사용합니다. 내부적으로 Self-RAG와 Corrective RAG의 아키텍처를 채택하여 보다 적절한 내용을 작성하고 환각 현상을 방지합니다.


사용 방법:

  • "단락 생성" 옵션을 선택합니다.
  • 작성하고 싶은 단락의 주제를 입력합니다.

    (예: "최근 삼성전자 관련 이슈를 정리해서 분석해줘")

  • 메시지를 전송하면 관련 정보를 포함한 단락을 생성합니다.
  • 예상 소요 시간: 30초 내외

예시:

  • 입력: "최근 삼성전자 관련 이슈를 정리해서 분석해줘"
  • 출력:

    • "최근 삼성전자는 HBM 기술과 관련된 이슈가 주목받고 있으며, 3분기 실적이 예상보다 저조해 주가가 5만 원대로 하락했습니다. 반도체 부문 영업이익은 6조 4500억 원으로 보고되었으나, 시장의 기대에 미치지 못했습니다. 또한, KB증권은 엔비디아향 HBM3E 공급 증가를 근거로 목표주가를 상향 조정했습니다."

요약 / 확장


텍스트 요약 및 확장 옵션은 사용자가 입력한 텍스트를 간결하게 줄이거나 상세히 확장하여 문맥에 맞는 정보를 제공합니다. 요약 기능은 긴 문장을 핵심만 남기고 축소하며, 확장 기능은 짧은 문장을 더 구체적으로 서술합니다.


사용 방법:

  • "요약 / 확장" 옵션을 선택합니다.
  • 요약하거나 확장하고 싶은 텍스트와 요약 / 확장 여부를 입력합니다.

    (예: "최근 삼성전자는 HBM 기술과 관련된 이슈가 주목받고 있으며, 3분기 실적이 예상보다 저조해... 확장해줘")

  • 메시지를 전송하면 입력된 텍스트를 간결하게 요약하거나 풍부하게 확장합니다. (단, 출처는 제공되지 않습니다.)
  • 예상 소요 시간: 5초 내외

요약:

  • 입력:

    • "삼성전자는 최근 HBM(High Bandwidth Memory) 기술과 관련된 여러 이슈로 인해 시장의 많은 주목을 받고 있습니다. 특히, 2023년 3분기 실적이 시장 예상에 미치지 못하면서 주가는 5만 원대까지 하락하는 상황에 직면... 이 문장을 요약해줘"

  • 출력: "최근 삼성전자는 HBM 기술과 관련된 이슈가 주목받고 있으며, 3분기 실적이 예상보다 저조해 주가가 5만 원대로 하락했습니다. 반도체 부문 영업이익은 6조 4500억 원으로 보고되었으나, 시장의 기대에 미치지 못했습니다. 또한, KB증권은 엔비디아향 HBM3E 공급 증가를 근거로 목표주가를 상향 조정했습니다."

확장:

  • 입력: "최근 삼성전자는 HBM 기술과 관련된 이슈가 주목받고 있으며, 3분기 실적이 예상보다 저조해... 이 문장을 확장해줘"
  • 출력:

    • "삼성전자는 최근 HBM(High Bandwidth Memory) 기술과 관련된 여러 이슈로 인해 시장의 많은 주목을 받고 있습니다. 특히, 2023년 3분기 실적이 시장 예상에 미치지 못하면서 주가는 5만 원대까지 하락하는 상황에 직면했습니다. 이와 관련하여 삼성전자의 반도체 부문 영업이익은 6조 4500억 원으로 보고되었으나, 이는 시장의 기대치를 크게 하회..."

데이터 시각화 (Web)


해당 옵션에서는 사용자가 요청한 시각화 자료를 웹에서 수집하여 생성합니다. 주로 그래프에 대한 데이터를 보유하고 있지 않을 때 유용하게 사용할 수 있습니다. 웹 검색을 통해 사용자 요청에 대한 자료를 수집하고 수집한 자료를 분석하여 적절한 그래프를 도식합니다. 그래프와 함께 수집한 데이터의 출처를 제공하여 사용자가 결과물을 검증하고 출처를 명시할 수 있도록 지원합니다.


사용 방법:

  • "데이터 시각화(Web)" 옵션을 선택합니다.
  • 시각화하고 싶은 주제를 입력합니다.

    (예: "최근 3개년 대한민국 GDP 추이 그려줘")

  • 메시지를 전송하면 사용자 요청에 대한 그래프를 생성합니다.
  • 예상 소요 시간: 2분 내외

데이터 시각화 (Upload)


해당 옵션에서는 사용자가 업로드한 CSV 데이터를 분석하여 시각화 자료를 생성합니다. 주로 그래프에 대한 데이터를 보유하여 다양한 분석과 시각화가 필요한 경우 유용하게 사용할 수 있습니다. 업로드한 자료를 다방면에 분석하고, 각 분석에 대한 적절한 그래프를 도식합니다. 그래프와 함께 제공되는 출처는 사용자가 업로드한 파일이 됩니다.


사용 방법:

  • "데이터 시각화(Upload)" 옵션을 선택합니다.

  • 예상 소요 시간: 2분 내외

FAQ (자주 묻는 질문)


Q1: 입력 텍스트는 어떤 형식이어야 하나요?

  • 텍스트나 간단한 키워드, 또는 데이터(파일 업로드)를 입력할 수 있습니다.

Q2: 지원되는 업로드 파일 형식과 최대 업로드 가능한 파일 수는 어떻게 되나요?

  • CSV, PDF파일 형식만 지원하며 최대 4개까지 가능합니다.

Q3: 생성된 결과물이 마음에 들지 않으면 어떻게 해야 하나요?

  • 입력 텍스트를 수정하거나 추가 정보를 명확히 입력한 뒤 다시 요청하세요.

Q4: 데이터 시각화는 어떤 종류의 차트를 생성하나요?

  • 막대 그래프, 꺾은선 그래프, 파이 차트 등 사용자의 데이터에 적합한 시각화를 제공합니다.
    웹 자료 기반 시각화의 경우 1개의 그래프가 생성되며, 업로드 데이터 기반 시각화의 경우 2개 이상의 다양한 시각화 자료를 제공합니다.
    업로드 데이터를 기반으로 그래프를 생성한 경우, 생성 직후 업로드한 CSV 파일은 서버에서 자동 삭제됩니다.
