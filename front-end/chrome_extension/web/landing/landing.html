FinPilot:
문서 관리의 프로페셔널

FinPilot은 Chrome 확장 프로그램으로 자체 개발한 sLLM을 사용하여 금융 문서 작성을 도와줍니다. 사용자의 Google Docs와 연동되어 실시간으로 답변 받고 이를 Google Docs에 바로 적용할 수 있습니다.

Key Features

  • 실시간 Google Docs 연동: AI 생성 콘텐츠를 문서에 바로 반영

  • 사용자 친화적 인터페이스 UI: 사이드 패널의 간단한 채팅으로 서비스 기능 제공

  • 효율적인 작업 흐름: AI 제안을 문서에 반영하며 생산성 향상

FinPilot만의 차별점

Google Docs 내에서 비즈니스 문서를 작성하고 편집할 수 있는 혁신적인 도구로, 직관적이고 강력한 사용자 경험을 제공합니다.

How to use?

시작

  1. Chrome 브라우저 환경에서 우측 상단의 FinPilot 아이콘을 클릭하고 로그인하세요.

  2. Google Docs에 접속 후 작업할 문서를 선택하세요.

  3. FinPilot 아이콘을 우클릭하고 "측면 패널 열기"를 클릭하세요.

  4. AI와 채팅을 통해 문서를 마음껏 작성하거나 편집하세요.

FinPilot의 놀라운 능력

  1. 원하는 기업이나 주제를 선정하여 분석 보고서의 초안을 마법처럼 생성합니다.


  2. 업로드 자료와 실시간 데이터를 결합하여 문맥에 맞는 정확한 문장을 작성하고 수정합니다.


  3. Google Docs와의 실시간 통합으로 팀원 간의 협업이 더욱 쉬워집니다.


  4. 복잡한 설정 없이 누구나 몇 번의 클릭으로 전문 문서를 작성할 수 있습니다.

FAQ

FinPilot은 어떤 방식으로 동작하나요?

AI에게 질문을 입력하면, FinPilot은 실시간으로 답변을 표시하며, 이를 작업 중인 구글docs에 반영하여 문서를 작성 및 편집할 수 있습니다.

어떤 종류의 문서를 생성할 수 있나요?

FinPilot은 금융과 관련된 용어를 잘 이해하고 있어, 특히 기업/시장 분석 보고서와 같은 작업에 특화되어 있습니다.

문서 ID를 찾을 수 없다는 오류가 나와요.

먼저 서비스 이용 전 로그인을 진행한 후, 현재 Chrome 활성탭이 Google Docs URL이 맞는지 확인해주세요.
