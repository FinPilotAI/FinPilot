초안 작성


초안 작성 옵션은 특정 주제, 질문 또는 키워드를 기반으로 초기 초안을 작성하는 데 유용합니다. 이는 아이디어를 구체화하거나 문서 작업의 시작점을 잡는 데 도움을 줍니다. 예를 들어, 새로운 프로젝트 제안서를 작성하거나 에세이 주제를 탐구할 때 적합합니다.


사용 방법:

  • "초안 작성" 옵션을 선택합니다.
  • 채팅창에 초안을 작성하고 싶은 주제를 입력합니다.

    (예: "친환경 에너지의 장점" 또는 "기술 발전이 경제에 미치는 영향")

  • 메시지를 전송하면 AI가 주제에 적합한 간략한 초안을 제공합니다.

예시:

  • 입력: "기후 변화의 심각성"
  • 출력:

    • "기후 변화는 전 세계적으로 주요한 문제로 떠오르고 있습니다. 이산화탄소와 같은 온실가스 배출은 지구 온난화를 가속화하며, 이는 생태계와 인간 생활에 광범위한 영향을 미칩니다. 지속 가능한 에너지 개발과 국제적인 협력이 필요한 시점입니다."

단락 생성


단락 생성 옵션은 사용자가 입력한 문장을 확장하여 더 상세한 내용을 포함한 단락을 작성합니다. 이는 논문, 보고서, 설명 자료 등을 작성할 때 정보의 깊이를 추가하는 데 유용합니다.


사용 방법:

  • "단락 생성" 옵션을 선택합니다.
  • 단락으로 확장하고 싶은 텍스트를 입력합니다.

    (예: "기술 발전의 장점")

  • 메시지를 전송하면 AI가 텍스트를 분석하고 관련 정보를 포함한 단락을 생성합니다.

예시:

  • 입력: "기술 발전의 장점"
  • 출력:

    • "기술 발전은 현대 사회의 생산성과 효율성을 극대화하는 데 기여하고 있습니다. 특히 인공지능과 빅데이터 기술은 기업이 더 나은 의사 결정을 내릴 수 있도록 도우며, 의료, 교육, 금융 등 다양한 분야에서 혁신적인 변화를 이끌고 있습니다."

요약 / 확장


텍스트 요약 및 확장 옵션은 사용자가 입력한 텍스트를 간결하게 줄이거나 상세히 확장하여 문맥에 맞는 정보를 제공합니다. 요약 기능은 긴 문장을 핵심만 남기고 축소하며, 확장 기능은 짧은 문장을 더 구체적으로 서술합니다.


사용 방법:

  • "요약 / 확장" 옵션을 선택합니다.
  • 요약하거나 확장하고 싶은 텍스트를 입력합니다.

    (예: "기술 발전은 현대 사회에 중요하다.")

  • 메시지를 전송하면 AI가 입력된 텍스트를 간결하게 요약하거나 풍부하게 확장합니다.

요약:

  • 입력:

    • "기술 발전은 현대 사회의 모든 영역에 영향을 미치며, 경제 성장과 사회 변화를 이끌어냅니다. 특히 인공지능과 로봇 기술은 생산성을 높이고 혁신을 가속화하는 데 중요한 역할을 하고 있습니다."

  • 출력: "기술 발전은 경제 성장과 혁신을 이끕니다."

확장:

  • 입력: "기술 발전은 중요하다."
  • 출력:

    • "기술 발전은 현대 사회에서 매우 중요한 역할을 합니다. 특히 인공지능, 빅데이터, 로봇 공학 등의 발전은 생산성 향상, 생활의 질 향상, 새로운 직업 창출 등 다양한 이점을 제공합니다."

데이터 시각화 (Web)


데이터 시각화(Web) 옵션은 사용자가 입력한 간단한 데이터를 기반으로 차트나 그래프를 생성합니다. 이 옵션은 빠른 데이터 분석과 의사결정을 돕기 위해 설계되었습니다.


사용 방법:

  • "데이터 시각화(Web)" 옵션을 선택합니다.
  • 시각화하고 싶은 데이터를 간단히 입력합니다.

    (예: "2023년 1월: 200, 2월: 250, 3월: 300")

  • 메시지를 전송하면 AI가 입력 데이터를 기반으로 그래프를 생성합니다.

예시:

  • 입력: "2023년 판매 데이터 (1월: 200, 2월: 250, 3월: 300)"
  • 출력: "월별 매출 데이터를 막대 그래프로 생성합니다."

데이터 시각화 (Upload)


업로드된 데이터를 분석하고 시각화합니다. 이 옵션은 Excel, CSV와 같은 파일 데이터를 기반으로 다양한 차트 및 그래프를 생성합니다. 주로 복잡한 데이터 분석에 사용됩니다.


사용 방법:

  • "데이터 시각화(Upload)" 옵션을 선택합니다.
  • 데이터를 포함한 파일(.PDF, .CSV)을 업로드합니다.
  • 메시지를 전송하면 AI가 데이터를 분석하고 적합한 시각화를 제공합니다.

예시:

  • 입력 파일: "2023년 분기별 매출 데이터.csv"
  • 출력: "분기별 매출 데이터를 꺾은선 그래프로 생성합니다."

FAQ (자주 묻는 질문)


Q1: 입력 텍스트는 어떤 형식이어야 하나요?

  • 텍스트나 간단한 키워드, 또는 데이터(파일 업로드)를 입력할 수 있습니다.

Q2: 지원되는 업로드 파일 형식은 무엇인가요?

  • CSV, PDF파일 형식을 지원합니다.

Q3: 생성된 결과물이 마음에 들지 않으면 어떻게 해야 하나요?

  • 입력 텍스트를 수정하거나 추가 정보를 명확히 입력한 뒤 다시 요청하세요.

Q4: 데이터 시각화는 어떤 종류의 차트를 생성하나요?

  • 막대 그래프, 꺾은선 그래프, 파이 차트 등 사용자의 데이터에 적합한 시각화를 제공합니다.
